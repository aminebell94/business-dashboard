# Product Overview

This is a business management dashboard application for managing orders, products, and customers. The system provides:

- Dashboard with KPIs (revenue, orders) and analytics charts
- Product catalog management (CRUD operations)
- Order management with line items and status tracking
- Customer information tracking
- Real-time analytics and reporting

The application connects to a Strapi CMS backend for data persistence and uses a modern React-based frontend for the user interface.

## Key Features

- Multi-status order workflow (preparation, delivery, completed, cancelled, etc.)
- Product inventory tracking with SKU and pricing
- Order line items with quantity and unit price calculations
- Dashboard analytics with revenue and order status charts
- French language support for order statuses and UI elements
